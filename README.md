# 111
再启航刷课略微修改版（来源于再启航自动刷课机作者VariantConst）

下面是教程
第一步：win＋R打开cmd，安装环境（依次输入以下命令）：
1. pip install -U selenium
2. pip install webdriver-manager
3. pip install --upgrade webdriver-manager

下面是需要的四个环境：
- Python
- Chrome浏览器（安装最新版chrome，我的版本是142.0.7444.176）
- Selenium
- ChromeDriver（下载地址：https://developer.chrome.google.cn/docs/chromedriver/downloads?hl=zh-cn）

然后：
1. 桌面新建文件夹，命名为 ZaiQiHang-AutoLearn-main
2. 将我的文件放入该文件夹

⚠️ 注意：运行后会自动打开chrome浏览器和网页，需要手动登录；刷课期间鼠标要一直放在视频上。
